# Inka Tatoo Landing

Landing page en Next.js inspirada en la estructura narrativa de `monasteriotattoo.com`, adaptada para la marca Inka Tatoo.

## Incluye

- Hero editorial con CTA y emblema SVG propio.
- Secciones de nosotros, servicios, trabajos, clientes, equipo, valores, FAQ y contacto.
- Ilustraciones SVG originales para el hero, la galería y el equipo.
- Carpeta `public/media/` lista para guardar video del hero e imágenes reales del estudio.
- Estilos responsivos en `app/globals.css`.

## Desarrollo

```bash
npm install
npm run dev
```

> Nota: en este entorno no fue posible descargar dependencias desde npm por una restricción del proxy. El código quedó preparado para ejecutarse localmente o en CI con acceso al registro.
